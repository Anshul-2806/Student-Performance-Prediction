# Student-Performance-Prediction

# 📚 Student Performance Prediction using Machine Learning

A complete Machine Learning project that predicts student marks based on the number of courses and study time using regression algorithms.

---

## 🚀 Features

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Multiple Regression Models
- Model Comparison
- Prediction System
- Streamlit Web App
- Model Saving with Joblib

---

## 📂 Dataset

Dataset contains:

- Number of Courses
- Study Time
- Marks

Target Variable:

- Marks

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Streamlit
- Joblib

---
